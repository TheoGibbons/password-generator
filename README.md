# Password Generator

Simple single-page HTML and JavaScript password generator.

# GIT
https://github.com/TheoGibbons/password-generator

# Demo
[https://theogibbons.github.io/password-generator/index.html](https://theogibbons.github.io/password-generator/index.html)

## Features

- Uppercase, lowercase, number, and symbol checkboxes
- Editable alphabet field
- Random password output that regenerates whenever inputs change
- Regenerate button
- Copy to clipboard button
- Password length slider from 1 to 50
- Password length number input for values longer than 50

## Usage

1. Open `index.html` in a browser.
2. Choose the character sets you want to use.
3. Edit the **Alphabet** field if you want a custom set of characters.
4. Set the password length with the slider or number input.
5. Copy the generated password with the **Copy to clipboard** button.
